개인정보 처리방침

최종 수정일: 2026년 2월 7일

1. 개요

Magic CMS(이하 "앱")는 디지털 사이니지 콘텐츠 관리 솔루션입니다. 본 개인정보 처리방침은 앱이 수집하는 정보와 그 사용 방법에 대해 설명합니다.

중요: Magic CMS는 사용자의 개인정보를 외부 서버로 전송하지 않습니다. 모든 데이터는 사용자의 기기에만 로컬 저장됩니다.

2. 수집하는 정보

앱은 서비스 제공을 위해 다음 정보를 수집합니다:

2.1 기기 정보

수집 항목 수집 목적 저장 위치
로컬 네트워크 기기 IP 주소 디지털 사이니지 기기 통신 로컬 기기
기기 이름 및 모델 기기 식별 및 관리 로컬 기기
네트워크 상태 정보 연결 상태 확인 임시 메모리

2.2 미디어 파일

  • 사용자가 선택한 이미지 및 동영상 파일
  • 프로그램 구성을 위한 콘텐츠 정보

2.3 카메라 접근

  • QR 코드 스캔을 통한 기기 등록 시에만 사용
  • 카메라로 촬영된 이미지는 저장되지 않음

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 기기 통신: 로컬 네트워크 내 디지털 사이니지 기기와의 통신
  • 콘텐츠 전송: 프로그램 및 미디어 파일을 기기로 배포
  • 기기 관리: 연결된 기기의 상태 모니터링 및 관리
  • 프로그램 저장: 사용자가 생성한 프로그램 정보 보관

4. 데이터 저장 및 보안

4.1 로컬 저장

모든 데이터는 사용자의 모바일 기기 내 SQLite 데이터베이스에 로컬 저장됩니다. 외부 클라우드 서버나 제3자 서버로 데이터가 전송되지 않습니다.

4.2 네트워크 통신

앱은 로컬 네트워크(WiFi/이더넷)에서 HTTP/WebSocket 서버로 동작하여 동일 네트워크 내 기기와만 통신합니다. 인터넷을 통한 외부 통신은 수행하지 않습니다.

4.3 데이터 삭제

앱을 삭제하면 저장된 모든 데이터가 함께 삭제됩니다.

6. 제3자 제공

Magic CMS는 수집된 정보를 제3자에게 판매, 거래, 임대하지 않습니다. 모든 데이터는 사용자의 기기 내에서만 처리됩니다.

1. Overview

Magic CMS (the "App") is a digital signage content management solution. This Privacy Policy explains what information the App collects and how it is used.

Important: Magic CMS does not transmit your personal information to external servers. All data is stored locally on your device only.

2. Information We Collect

The App collects the following information to provide its services:

2.1 Device Information

Data Collected Purpose Storage Location
Local network device IP addresses Digital signage device communication Local device
Device name and model Device identification and management Local device
Network status information Connection status verification Temporary memory

2.2 Media Files

  • Images and video files selected by the user
  • Content information for program configuration

2.3 Camera Access

  • Used only for device registration via QR code scanning
  • Images captured by the camera are not stored

3. How We Use Information

Collected information is used only for the following purposes:

  • Device Communication: Communication with digital signage devices within the local network
  • Content Delivery: Deploying programs and media files to devices
  • Device Management: Monitoring and managing connected device status
  • Program Storage: Storing program information created by users

4. Data Storage and Security

4.1 Local Storage

All data is stored locally in an SQLite database on the user's mobile device. No data is transmitted to external cloud servers or third-party servers.

4.2 Network Communication

The App operates as an HTTP/WebSocket server on the local network (WiFi/Ethernet) and communicates only with devices on the same network. No external communication over the internet is performed.

4.3 Data Deletion

Uninstalling the App will delete all stored data.

5. App Permissions

The App requests the following permissions to provide its services:

Permission Purpose
Camera Device registration via QR code scanning
Photo Library Selecting images/videos for program creation
Microphone Audio capture during video recording
Local Network Communication with digital signage devices
WiFi State Network connection status check

6. Third-Party Sharing

Magic CMS does not sell, trade, or rent collected information to third parties. All data is processed only within the user's device.

7. Children's Privacy

This App is not directed at children under 13 years of age, and we do not knowingly collect personal information from children under 13.
