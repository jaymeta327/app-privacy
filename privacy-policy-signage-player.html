1. 개요

Magic Signage Player(이하 "앱")는 디지털 사이니지 콘텐츠 플레이어 앱입니다. 본 앱은 무료로 제공되는 Magic CMS 앱(스마트폰)과 연동하여 Android 셋톱박스 및 Android TV 셋톱박스에서 사이니지 콘텐츠를 재생합니다. 본 개인정보 처리방침은 앱이 수집하는 정보와 그 사용 방법에 대해 설명합니다.

중요: Magic Signage Player는 사용자의 개인정보를 외부 서버로 전송하지 않습니다. 모든 데이터는 사용자의 기기에만 로컬 저장되며, 동일 네트워크 내 Magic CMS 앱과만 통신합니다.

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 라이센스 인증: 기기 식별자를 사용한 영구 라이센스 인증
  • 기기 통신: 로컬 네트워크 내 Magic CMS 앱(스마트폰)과의 통신
  • 콘텐츠 수신: Magic CMS에서 전송되는 사이니지 콘텐츠 수신 및 저장
  • 콘텐츠 재생: 이미지, 동영상(4K), 웹페이지, YouTube 등 콘텐츠 재생
  • 스케줄 관리: 시간대별 자동 콘텐츠 전환

4. 데이터 저장 및 보안

4.1 로컬 저장

모든 데이터는 셋톱박스 기기 내 SQLite 데이터베이스 및 로컬 저장소에 저장됩니다. 외부 클라우드 서버나 제3자 서버로 데이터가 전송되지 않습니다.

4.2 네트워크 통신

앱은 로컬 네트워크(WiFi/이더넷)에서 동일 네트워크 내 Magic CMS 앱과만 통신합니다. 인터넷을 통한 외부 서버 통신은 수행하지 않습니다.

4.3 데이터 삭제

앱을 삭제하면 저장된 모든 데이터(콘텐츠, 설정, 라이센스 정보)가 함께 삭제됩니다.

1. Overview

Magic Signage Player (the "App") is a digital signage content player application. This App works with the free Magic CMS app (smartphone) to play signage content on Android set-top boxes and Android TV set-top boxes. This Privacy Policy explains what information the App collects and how it is used.

Important: Magic Signage Player does not transmit your personal information to external servers. All data is stored locally on your device only and communicates only with the Magic CMS app on the same network.

2. Information We Collect

The App collects the following information to provide its services:

2.1 Device Information

Data Collected Purpose Storage Location
Device identifier License authentication and device registration Local device
Local network device IP addresses Communication with Magic CMS app Local device
Device name and model Device identification and management Local device
Network status information Connection status verification Temporary memory

2.2 Media Files

  • Signage content (images, videos) received from Magic CMS
  • Media files from USB storage (USB mode)
  • Content information for program configuration

3. How We Use Information

Collected information is used only for the following purposes:

  • License Authentication: Permanent license authentication using device identifier
  • Device Communication: Communication with Magic CMS app (smartphone) within the local network
  • Content Reception: Receiving and storing signage content transmitted from Magic CMS
  • Content Playback: Playing images, videos (4K), web pages, YouTube, and other content
  • Schedule Management: Automatic content switching by time schedule

4. Data Storage and Security

4.1 Local Storage

All data is stored in an SQLite database and local storage on the set-top box device. No data is transmitted to external cloud servers or third-party servers.

4.2 Network Communication

The App communicates only with the Magic CMS app on the same local network (WiFi/Ethernet). No external server communication over the internet is performed.

4.3 Data Deletion

Uninstalling the App will delete all stored data (content, settings, and license information).

5. App Permissions

6. Third-Party Sharing

Magic Signage Player does not sell, trade, or rent collected information to third parties. All data is processed only within the user's device.
